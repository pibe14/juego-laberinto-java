# Juego ejecutado en consola
Minijuego en java para ejecutar en consola que consta de 3 niveles con enemigos que se mueven

### Para ejecutarlo
En la terminal de tu ordenador escribes los siguientes comandos desde el **directorio raíz del proyecto**:

_Para compilar y ejecutar_
```bash
javac -d target/classes src/main/java/juego_java/*.java
cd target/classes
java juego_java.Main
```

_Para compilar y ejecutar con **Maven**_
```bash
mvn compile
cd target\classes
java juego_java.Main
```
